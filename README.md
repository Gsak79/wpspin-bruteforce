# wpspin-bruteforce
A Bash script that automates WPS PIN brute-force attempts by generating candidate PINs with wpspin and testing them using reaver.

## Use
```bash
git clone https://github.com/Gsak79/wpspin-bruteforce.git
cd wpspin-bruteforce
chmod +x wpspin-bruteforce.sh
sh wpspin-bruteforce.sh <BSSID> <CHANNEL> <INTERFACE>
```
